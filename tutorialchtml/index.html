Variáveis e constantes:

Uma variável é um espaço de memória que pode ser identificado por um nome antes atribuido a ele pelo programador.

Pode ter seu conteúdo alterado ao longo do programa por um comando de atribuição, o valor antigo que ocupava o espaço na memória é apagado e agora aquela variável se refere somente ao novo valor, podendo ocorrer ilimitadas vezes.

Exitem algumas regras na identificação de variáveis: Os caracteres que você pode utilizar na formação dos identificadoes são: números, letras maiúsculas, letras minúsculas e o underscore (_). O primeiro caractere deve ser sempre uma letra, e palavras reservadas da linguagem C não podem ser identificadores.

Os 4 tipos principais de variáveis, sendo eles:

  • Caractere - char (a, b, c, 1, 2, #, ^, * )
  • Inteiro - int ( -2, -1, 0, 1, 2 )
  • Real - float ( 1.2, 2.5, -20.456 )
  • Lógico - int ( A variável lógica não existe em C, para representá-la usamos int com 0 para falso, e qualquer número inteiro para verdadeiro)

Já as constantes, devem ser declaradas antes do bloco principal (main), para atribuir um valor a uma constante digitamos:
#define "constante" "valor"

Os operadores aritiméticos básicos são os de:

  • Adição - ( + )
  • Subtração - ( - )

Operadores relacionais:

Os operadores relacionais são usados em conjuto com os operandos, que podem ser variáveis ou constantes já declaradas. Juntos, formam expressões lógicas que retornam valores de verdadeiro ou falso.

Os operadores relacionais são:

  • Igual - ( == )
  • Maior - ( > )
  • Menor - ( < )
  • Maior ou igual - ( >= )
  • Menor ou igual - ( <= )
  • Diferente - ( != )

Estruturas condicionais:

Chamamos de estruturas condicionais as instruções para testar se uma condição é verdadeira ou não.

A instrução if é a estrutura de teste mais básica. Ela permite executar uma série de instruções caso uma condição se realize. A sintaxe desta expressão é a seguinte: 

 	
 	if (condição a ser testada)
 	{
		lista de instruções;
	}

Se a condição testada for verdadeira, o computador segue a lista de intruções, se não, a mesma é ignorada e o computador pula para o próximo item.

A condição deve ficar entre parênteses. É possível definir várias condições a serem cumpridas com os conectores E e OU ( && e ||). Para escolher as instruções a serem executadas em caso do não cumprimento da condição, usamos a expressão else. 

 	
 	if (condição a ser testada)
 	{
		lista de instruções;
	}
	else
	{
		lista de instruções(2);
	}

A instrução switch efetua vários testes de valores sobre o conteúdo de uma mesma variável. Esta estrutura condicional simplifica muito o teste de vários valores de uma variável, a operação também pode ser feita com if e else, porém ficaria maior e mais complicada. Sua sintaxe é a seguinte: 

 	
	switch (variável) {

		caso Valor1:
		lista de instruções;
		break;

		caso Valor2 :
		lista de instruções;
		break;

		caso Valores... :
		lista de instruções;
		Break;

		default:
		lista de instruções;
}

Operações de entrada e saída de dados:

Chamamos de saída de dados a exibição de textos ou valores de variáveis no vídeo.
Para a exibição de dados na tela, usamos a função printf.
Não há regras para o que pode ser exibido dentro de um comando printf.

Já a entrada de dados, recolhe os dados informados pelo usuário, que pode por exemplo ser um texto ou valores.
Para o recolhimento desses dados usamos a função scanf.
Para definir o tipo de dado que esta sendo recolhido ou que será exibido, usamos os indicadores:

  • Int - %i
  • Float - %f
  • Char - %c
  • String - %s
